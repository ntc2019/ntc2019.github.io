HTML Cheatsheet
HTML Element Example code

<table> Table Element
The table element is used to organize data into rows and columns. 

<table>
    .....
    <!--table data goes here-->
</table>

<thead> Table head element
The table head element <thead> define the header of a table.

 
<table>
    <thead>
    <!--header content-->
        <tr>
            <th>
                Name
            </th>
            <th>
                Adress
            </th>
        </tr>
    </thead>
</table>
<tbody> Table body element
The table <tbody> body element indicate the data content of a table. 
<table>
    <thead>
    </thead>
    <tbody>
        <tr>
            <td>
                Richard
            </td>
            <td>
                07,Beverly Hill
            </td>                                                                      
            <td>
                Johnson
            </td>
            <td>
                105, Hampton Ville
            </td>                                                                      
        </tr>
    </tbody>
</table>
<tr> Table row element
The table row, <tr>, is used to add rows to table before adding table data and table headings. 
<table>
    <tr>
        ....
    </tr>
<table/>
<td> Table Data Element
The table data element, <td>, can be nested inside a table row element, <tr>, to add a cell of table data to a table. 
<table>
    <tr>
        <td>cell one data</td>
        <td>cell two data</td>
    </tr>
</table>

<tfoot> Table Footer Element
The table footer element, <tfoot>, uses table rows to give footer content or to summarize content at the end of the table.

 
<table>
    <thead>
        <tr>
            <th>Headings one</th>
            <th>Headings two</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data Column one</td>
            <td>Data Column two</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Summary of column one</td>
            <td>Summary of column two</td>
        </tr>
    </tfoot>
</table>

colspan Attribute
The colspan attribute on a table header <th> or table data <td> element indicates how many columns that particular cell should span within the table. The colspan value is set to 1 by default and will take any positive integer between 1 and 1000. 

<table>
    <tr>
        <th>Row 1:</th>
        <td>col 1</td>
        <td>col 2</td>
        <td>col 3</td>
    </tr>
    <tr>
        <th>Row 2:</th>
        <td colspan="2">col 1 + 2 (will span 2 columns)</td>
        <td<col 3</td>
    </tr>
</table>

rowspan Attribute
Similar to colspan, the rowspan attribute on a table header or table data element indicates how many rows that particular cell should span within the table. The rowspan value is set to 1 by default and will take any positive integer up to 65534. 

<table>
    <tr>
        <th>Row 1:</th>
        <td>col 1</td>
        <td>col 2</td>
    </tr>
    <tr>
        <th rowspan="2">Row 2 + 3:</th>
        <td>col 1</td>
        <td>col 2</td>
    </tr>
    <tr>
        <td>col 1</td>
        <td>col 2</td>
    </tr>
</table>
